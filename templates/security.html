About Roles
  • Super Admin (Level 4) — Always has all permissions. Cannot be edited.
  • Admin (Level 3) — Can manage guests, hosts, users. Sees all guests regardless of tags.
  • Operator (Level 2) — Can SSH and apply updates on assigned guests.
  • Viewer (Level 1) — Read-only access to assigned guests.
Custom roles inherit a level from their base tier, which determines hierarchy. Users can only manage other users with a lower role level. Permissions on built-in roles (except Super Admin) can be customized.
